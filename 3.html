Linux

O Linux é um sistema operacional de código aberto amplamente reconhecido por sua versatilidade, estabilidade e robustez. Diferentemente de sistemas operacionais comerciais, o Linux permite que os usuários acessem, modifiquem e distribuam seu código-fonte, promovendo uma comunidade ativa e colaborativa.

Características

  1. Código Aberto: O Linux é baseado em software de código aberto, permitindo que o código-fonte seja acessado, modificado e redistribuído livremente.
  2. Segurança: Conhecido por sua robusta arquitetura de segurança, o Linux implementa permissões e separação de privilégios para proteger contra ameaças.
  3. Variedade de Distribuições: Há uma diversidade de distribuições Linux, cada uma adaptada para atender a necessidades específicas, proporcionando flexibilidade de escolha aos usuários.
  4. Desempenho e Estabilidade: O Linux é reconhecido por sua estabilidade e eficiência de desempenho, especialmente em ambientes de servidor.
  5. Comunidade Ativa: A comunidade Linux é ativa e colaborativa, oferecendo suporte através de fóruns, wikis e canais de IRC, proporcionando um ambiente de aprendizado e resolução de problemas

Principais versões

  • Ubuntu: Reconhecida por sua facilidade de uso, ampla comunidade e ciclos regulares de lançamento, o Ubuntu é uma escolha popular para usuários iniciantes e avançados.
  • Fedora: Focada em tecnologias de ponta, o Fedora é conhecido por ser uma distribuição inovadora que serve como base para o desenvolvimento do Red Hat Enterprise Linux (RHEL)
  • Debian: Conhecido por sua estabilidade e compromisso com o software livre, o Debian é utilizado como base para muitas outras distribuições, incluindo o Ubuntu.
  • CentOS: Baseado no código-fonte do Red Hat Enterprise Linux, o CentOS é uma escolha comum para servidores devido à sua estabilidade e suporte de longo prazo.
  • Arch Linux: Destinado a usuários avançados, o Arch Linux é uma distribuição minimalista que permite uma instalação altamente personalizada, adaptada às preferências individuais.
  • Linux Mint: Projetada para ser amigável e fácil de usar, o Linux Mint é uma distribuição baseada no Ubuntu e Debian, com foco na simplicidade e na experiência do usuário.

Exigências do sistema operativo quanto às capacidades do equipamento

  • Os requisitos podem variar significativamente dependendo da distribuição Linux escolhida. Distribuições leves podem operar eficientemente em hardware mais antigo, enquanto distribuições mais robustas podem ter requisitos mais alt

Instalação do SO a partir de periféricos

Instalação do Linux a partir de uma Unidade USB:

Passo 1: Preparação da Unidade USB

  1. Baixe o Ubuntu:
    • Vá para o site oficial do Ubuntu e baixe a versão desejada.
  2. Crie uma Unidade USB Inicializável:
    • Baixe o aplicativo Rufus (para Windows) ou Etcher (para macOS e Linux).
    • Execute o aplicativo e selecione a unidade USB.
    • Selecione a imagem ISO do Ubuntu baixada.
    • Clique em "Iniciar" ou "Gravar" para criar a unidade USB inicializável.

Passo 2: Configuração do Computador para Inicialização a partir da USB

  1. Reinicie o Computador:
    • Reinicie o computador e acesse o menu de inicialização (geralmente pressionando F2, F10, F12 ou DEL durante a inicialização).
  2. Configure a Ordem de Inicialização:
    • No menu de inicialização, configure a ordem de inicialização para iniciar a partir da unidade USB.

Passo 3: Iniciar a Instalação do Linux (Ubuntu)

  1. Inicie a partir da Unidade USB:
    • Salve as configurações e saia do BIOS. O computador deve iniciar a partir da unidade USB.
  2. Escolha "Instalar Ubuntu":
    • Na tela de inicialização do Ubuntu, escolha "Instalar Ubuntu".
  3. Selecione as Configurações de Idioma e Teclado:
    • Escolha suas configurações de idioma e teclado e clique em "Continuar".
  4. Escolha o Tipo de Instalação:
    • Selecione "Instalação Normal" ou "Instalação Mínima", dependendo das suas preferências.
  5. Selecione a Partição:
    • Escolha a partição onde deseja instalar o Ubuntu e clique em "Instalar Agora".
  6. Configuração Inicial:
    • Siga as instruções para configurar seu idioma, conta de usuário, senha, entre outros.
  7. Aguarde a Instalação:
    • Aguarde enquanto o Ubuntu é instalado no seu computador.
  8. Conclua a Instalação
    • Após a conclusão, reinicie o computador e retire a unidade USB.
